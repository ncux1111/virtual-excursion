# Виртуальная экскурсия

Офлайн-приложение по творчеству М. Е. Салтыкова-Щедрина. Проект содержит интерактивный маршрут, страницы остановок, задания для учащихся и поддержку локальных видео.

Приложение написано на React и TypeScript, собирается через Vite. Для desktop-версии используется Electron.

## Установка

Нужен Node.js и npm.

```bash
npm install
```

## Запуск в браузере

```bash
npm run dev
```

После запуска Vite откроет локальный адрес, обычно `http://localhost:5173`.

## Запуск Electron

```bash
npm run electron:dev
```

Эта команда запускает Vite dev server и открывает приложение в окне Electron.

## Сборка web-версии

```bash
npm run build
```

Готовая web-сборка создается в папке `dist/`. Эта папка не хранится в репозитории, потому что ее можно собрать заново из исходников.

## Сборка Windows-приложения

```bash
npm run electron:build
```

Готовый `.exe` создается в папке `release/`. Папка `release/` и файлы `.exe` не входят в репозиторий. Их лучше загружать отдельно в GitHub Releases.

## Видео и тяжелые файлы

Видео должны лежать в папке `public/videos/`:

- `stop-1.mp4`
- `stop-2.mp4`
- `stop-3.mp4`
- `stop-4.mp4`

Папка `public/videos/` не включена в репозиторий, потому что видео занимают много места. Для публикации видео используйте GitHub Releases или Git LFS.

Если видео отсутствуют, приложение не падает: на странице остановки будет показан placeholder.

## Что не входит в репозиторий

В Git не добавляются:

- `node_modules/` - зависимости, устанавливаются через `npm install`;
- `dist/` - результат web-сборки;
- `release/` - результат Electron-сборки;
- `public/videos/` - тяжелые видеофайлы;
- `.env` и другие локальные файлы окружения;
- логи, кэш, временные файлы;
- архивы и установщики: `.zip`, `.rar`, `.7z`, `.exe`, `.msi`.
